# PakExplorer

PakExplorer is a simple Quake `.pak` and `.pk3` archive viewer inspired by PakScape and originally developed by Peter Engstr√∂m. It renders the archive directory tree, lets you add/remove files, and export modified archives.

## Building

Use Xcode (macOS) to open `PakExplorer.xcodeproj` and build the project. There is also a [GitHub Actions workflow](.github/workflows/build.yml) that compiles the app on `macos-latest` and uploads a release artifact.
